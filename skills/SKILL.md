# Skill: Example Desktop Task

## Description

Demonstrates how to write a Caelum-Agent skill using the OpenClaw/Claude Code skill format.

## Usage

```
Run the example desktop task
```

## Steps

1. Take a screenshot to understand the current desktop state.
2. Identify the target application window.
3. Click the target element.
4. Verify the expected outcome.

## Notes

- Keep steps deterministic.
- Prefer accessibility trees over coordinate-based actions.
- Add verification after each action.
